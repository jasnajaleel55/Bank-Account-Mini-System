Bank Account Mini-System

This project is a simple Python program that shows a basic bank account system.
It allows the user to deposit money, withdraw money, and check the current account balance.

Objective :
To build a mini banking system using functions and combining multiple function types in one small system.

function:
balance variable

 deposit(amount):
Adds the specified amount to the account balance.

 withdraw(amount):
checks if the balance is sufficient
Deducts the amount if enough balance is available
Otherwise prints "no sufficient balance!"

 check_balance():
Displays the current balance.


Menu System
The program repeatedly shows this menu:
1. Deposit
2. Withdraw
3. Check Balance
4. Exit

The loop continues until the user selects Exit.
